一款纯本地的暗黑4事件倒计时软件,不读取修改内存,不联网,只是一个显示在暗黑4游戏画面顶层的专用倒计时时钟.

✨ 核心功能：

✅ 完全支持国服以及外服  
✅ 全透明悬浮窗，鼠标完全穿透，不影响游戏操作  
✅ 实时显示红潮/世界Boss/军团事件倒计时  
✅ 智能语音提醒（5分钟/2分钟预警 + 事件开始/结束播报）  
✅ 视觉闪烁预警（倒计时≤2分钟时白红交替闪烁）  
✅ Ctrl+F12 一键切换拖动模式，自由调整位置  
✅ 支持时间校准，右键托盘图标即可调整基准时间  
✅ 自动适配 1080P/2K/4K 及任意 DPI 缩放  
✅ 兼容窗口/无边框全屏/独占全屏所有模式  

━━━━━━━━━━━━━━━━━━━━

📖 使用方法：  
1️⃣ 下载后直接运行 exe 文件（无需安装）  
2️⃣ 启动游戏  
3️⃣ 悬浮窗自动显示在左上角，默认透明穿透  
4️⃣ 如需调整位置：按 Ctrl+F12 显示标题栏，按住拖动  
5️⃣ 如需校准时间：右键托盘图标 → 设置  可以在暗黑核等网站查询到下一次事件开始的时间,填入即可
